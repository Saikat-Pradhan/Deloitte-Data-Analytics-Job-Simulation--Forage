# 📊 Deloitte Data Analytics Job Simulation – Forage

This repository documents my completion of the Deloitte Data Analytics Job Simulation on Forage.
The project showcases practical applications of Tableau and Excel for telemetry analysis, downtime tracking, and employee compensation equality scoring.

---

## 🚀 Overview

- Conducted telemetry data analysis in Tableau to measure downtime across factories and device types.
- Built interactive dashboards to filter and compare downtime metrics.
- Processed employee compensation equality data in Excel, classifying fairness levels based on equality scores.
- Delivered structured outputs for both tasks, demonstrating skills in data visualization, reporting, and storytelling.

---

## Task 1: Telemetry Data Analysis (Tableau)

### Steps:
- Install Tableau – Download the free trial and register with your email.
- Import Dataset – Unzip daikibo-telemetry-data.json.zip and import into Tableau, ensuring all schema levels are checked.
- Create Calculated Field – Add a measure field called “Unhealthy” with a value of 10 for every unhealthy status (representing 10 minutes downtime).
- Visualizations:
   - Bar chart: Down Time per Factory
   - Bar chart: Down Time per Device Type
- Dashboard – Combine both charts into a dashboard. Set the first chart (Factory) as a filter for the second (Device Type).
- Submission – Select the factory with the most downtime, take a screenshot of the dashboard, and upload.

## Task 2: Equality Score Classification (Excel)

### Dataset:
- Factory
- Job Role
- Equality Score (integer, ranging from -100 to +100; 0 is ideal)

### Steps:
- Open Equality Table.xlsx.
- Add a 4th column: Equality Class.
- Classify scores into categories:
   - Fair → between -10 and +10
   - Unfair → between -20 and -11 OR between +11 and +20
   - Highly Discriminative → less than -20 OR greater than +20
- Save and upload the edited file.
